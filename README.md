# JWT Authuntication with role Based Access and Page Permissions
--> node version 18.17.0
-->project requirment : - Implement an authentication system where users can register, log in, and log out securely.
- Implement authorization based on roles (e.g., Admin, User, Moderator). Ensure that each role has specific permissions to access certain resources or endpoints.
- Use secure methods such as JWT or OAuth for managing sessions and user authentication.
- Implement **Role-Based Access Control (RBAC)**, where the access to resources is determined based on the user's assigned role.

- Steps to run Project :
- run npm install
- npm run dev
- add new role with pagePermissions
- create user
- get Users after login successfully

- implemented using typescript, mongoDB.
- complete robust api implementation with code reusability and followed MVC architecture.
